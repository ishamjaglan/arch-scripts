# Setup Script
This script can make your simple arch distro to a hacking distro in a few steps

## Usage:
	
	git clone https://github.com/ixh4m/arch-scripts.git
	cd arch-scripts/hacking-setup
	sudo chmod +x setup.sh
	./setup.sh
